## Description
This shell script automatically organizes files from the `~/Downloads` directory
into categorized subdirectories inside a new `~/Organized_Files` folder.
It sorts files by type (Images, Documents, Videos, Archives), logs the activity
with a timestamp, and prints a summary to the console.

The script opens by printing a stanza from **"Horatius at the Bridge"** by
Thomas Babington Macaulay — a reminder that even a humble file organizer
deserves a heroic send-off before facing the chaos of a Downloads folder.
